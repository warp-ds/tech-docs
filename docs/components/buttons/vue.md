## Import

> Use in entire app

```js
import { Button } from "@warp-ds/vue";
app.use(Button);
```

> Use in one component and special imports

```js
import { wButton } from "@warp-ds/vue";
```

## Syntax

```js
<w-button primary>Click me</w-button>
<w-button>Click me</w-button>
```
