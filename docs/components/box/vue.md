### Import

> Use in entire app

```js
import { Box } from "@warp-ds/vue";
app.use(Box);
```

> Use in one component and special imports

```js
import { wBox } from "@warp-ds/vue";
```

### Syntax

```vue
<w-box info>
    <p>This is <strong>info</strong> variant of the box component</p>
</w-box>
```
