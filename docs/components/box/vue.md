## Import

> Use in entire app

```js
import { Box } from "@warp-ds/vue";
app.use(Box);
```

> Use in one component and special imports

```js
import { wBox } from "@warp-ds/vue";
```

## Syntax

```js
<w-box info>
    <h1>Default example</h1>
    <p>This is "info" variant of the box component</p>
</w-box>
```
